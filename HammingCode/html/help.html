Использование программы

1. Задание параметров эксперимента

Задайте параметры эксперимента в боковой панели слева:

  • Число итераций — число раз, сколько будет передаваться сигнал в рамках эксперимента. На каждой итерации на отправленный сигнал накладывается новая помеха (целое положительное).

Код Хэмминга

Задайте последовательность и параметры кодирования:

  • Последовательность — последовательность из 0 и 1, которая будет закодирована и отправлена. Длина последовательности должна быть кратна размеру блока (см. ниже).
  • Размер блока — размер блока закодированной последовательности (целое положительное).
  • Модифицированный — использовать ли модифицированную версию кода Хэмминга.

Сигнал

Задайте параметры преобразования закодированной последовательности в сигнал:

  • Метод преобразования сигнала — доступно 4 метода преобразования сигнала:
    • NRZ — Non-Return-to-Zero
    • Манчестерский код
    • RZ — Return-to-Zero
    • AMI — Alternate Mark Inversion
  • Временной интервал — расстояние между двумя соседними точками при передаче одного бита (вещественное положительное)
  • Амплитуда (вещественное положительное)
  • Интервалов на бит — количество интервалов, используемое для передачи одного бита
    • (для NRZ, RZ AMI: вещественное положительное)
    • (для манчестерского кода: вещественное положительное четное)
  • Полярность — параметр, который определяет, каким уровнем (высоким или низким) представляется единица или ноль в сигнале (прямая или инвертированная)

Помеха

Задайте параметры импульсного шума, который будет наложен на сигнал:

  • Длительность выбирается в [t-Δt, t+Δt] (t, Δt — вещественные неотрицательные)
  • Частота повторения выбирается в [ν-Δν, ν+Δν] (ν, Δν — целые неотрицательные)
  • Полярность — параметр, определяющий, в каком направлении помеха может искажать сигнал (однополярная или биполярная)
  • Импульс имеет прямоугольную форму a̅, где a̅ выбирается в [a-Δa, a+Δa] (a — вещественное, Δa — вещественное неотрицательное)

2. Проведение эксперимента

Для проведения экспериментов нажмите на кнопку Расчёт, расположенную под панелью параметров помехи. Для выполнения первых четырех расчетов необходимо выполнить задание. Счетчик выполненных заданий сбрасывается при перезапуске программы.

Задания делятся на задачи кодирования и декодирования. Для каждого задания указано, является ли код Хэмминга модифицированной версией, и дана последовательность для работы. Для задач декодирования с модифицированным кодом Хэмминга требуется указать вердикт проверочного бита.

Введите ответ на задание в поле Ответ (и выберите Вердикт в случае задачи декодирования с модифицированным кодом Хэмминга) и нажмите Проверить. Число попыток не ограничено.

3. Просмотр результатов

В нижней части боковой панели (слева) представлена следующая информация:

  • Закодированная последовательность
  • Доверительный интервал и уровень доверия

Графики сигналов

В центральной части окна расположена область для просмотра графиков. Выбор графика для просмотра осуществляется на верхней панели, доступны следующие опции:

  • Отправляемый сигнал
  • Принимаемый сигнал с наибольшей ошибкой
  • Принимаемый сигнал с наименьшей ошибкой
  • Принимаемый сигнал с медианной ошибкой
  • Принимаемый сигнал с наибольшей верно исправленной ошибкой
  • Принимаемый сигнал на выбранной итерации

Управление в области графика:

  • ЛКМ — перемещение
  • ПКМ — просмотр положения курсора в координатах графика
  • Колесико мыши — приближение/отдаление

Кнопка Копировать в буффер, расположенная в правой части верхней панели, копирует в буффер обмена видимую область графика в повышенном качестве (1920x1080 или эквивалентное для текущего соотношения сторон графика).

Таблица итераций

Кнопка Показать таблицу, расположенная в правой части верхней панели, открывает окно с таблицей, содержащей информацию о каждой итерации:

  • Принятая последовательность
  • Восстановленная последовательность
  • Декодированная последовательность
  • Количество искажённых помехой бит
  • Количество верно исправленных бит в закодированной последовательности
  • Количество верно исправленных бит в декодированной последовательности
  • Количество ошибок в декодированной последовательности
  • Доля ошибок в декодированной последовательности
  • Количество верных срабатываний проверочного бита (если используется модифицированная версия кода Хэмминга)

Строки таблицы загружаются не сразу, поэтому по мере прокрутки могут появляться новые данные.

Для копирования данных из таблицы выберите желаемую область при помощи ЛКМ или нажмите Ctrl+A для выделения всех загруженных данных, а затем выберите соответствующую опцию в контекстном меню (ПКМ) или нажмите Ctrl+C.
